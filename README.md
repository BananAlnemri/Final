## 🌟 رؤية المشروع
يهدف المشروع إلى ردم فجوة التواصل بين الأطفال الصم والمجتمع من خلال تحويل الهجاء الإصبعي العربي إلى لغة حية مفهومة، وتعليم الأطفال السليمين لغة الإشارة بطريقة ممتعة وتفاعلية (Gamification).

## 🧠 مميزات التطبيق (الذكاء الاصطناعي)
* **Sign-to-Text:** ترجمة فورية لـ 32 فئة من حروف الهجاء العربي باستخدام الكاميرا.
* **Speech-to-Sign:** تحويل نطق الطفل إلى سلسلة من الصور الإشارية التعليمية (Voice-driven learning).
* **Real-time Feedback:** نظام ذكي يعطي تلميحات فورية للطفل لتصحيح وضعية اليد.
* **Gamification:** نظام نقاط، أوسمة، ولوحة صدارة (Leaderboard) لتحفيز الأطفال.

## 🛠 البناء التقني (Tech Stack)
* **الذكاء الاصطناعي (AI):**
    * `MobileNetV2` (Transfer Learning).
    * `TensorFlow / Keras`.
    * Dataset: 54,049 صورة للحروف العربية.
* **الواجهات (Frontend):** [:Fluttert].
* **معالجة اللغات (NLP):**
    * تحويل الكلام إلى نص (Speech-to-Text).
    * تقطيع الكلمات وعرض تسلسل الحروف (Text-to-Sign Mapping).

## 📱 واجهات التطبيق
1.  **رفيق الرحلة:** اختيار شخصية كرتونية تفاعلية تشجع الطفل.
2.  **مسار الحروف والكلمات:** تعلم الهجاء حرفاً بحرف.
3.  **مساحة التحدي:** تحديات زمنية لجمع النقاط.
4.  **المستكشف الذكي:** تحدث بكلمة، وشاهد كيف تُكتب بلغة الإشارة.
